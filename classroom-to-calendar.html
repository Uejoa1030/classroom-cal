Google Classroom → Google Calendar

1 設定 Google API

需要先在 Google Cloud Console 建立專案，啟用 Google Classroom API 和 Google Calendar API， 然後建立 OAuth 2.0 憑證（類型選「網頁應用程式」），授權的 JavaScript 來源填入此頁面的網址（例如 http://localhost 或 GitHub Pages 網址）。 詳細步驟見下方說明。

📖 設定教學

? 申請 API 金鑰教學

  1. 前往 Google Cloud Console 並建立新專案
  2. 左側選單 → API 和服務 → 程式庫
  3. 搜尋並啟用 Google Classroom API
  4. 再次搜尋並啟用 Google Calendar API
  5. 左側選單 → 憑證 → 建立憑證 → API 金鑰（複製備用）
  6. 再次建立憑證 → OAuth 用戶端 ID → 類型選「網頁應用程式」
  7. 「已授權的 JavaScript 來源」填入此頁面開啟的網址
  8. 複製 用戶端 ID 和 API 金鑰 填入上方欄位
  9. 左側 → OAuth 同意畫面，測試使用者加入你的 Google 帳號
